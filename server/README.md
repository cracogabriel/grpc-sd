# mflix Python Server

A gRPC server that handles movie management operations on a MongoDB Atlas database.  
Receives requests from the Java client using Protocol Buffers and gRPC.

---

## Requirements

- Python 3.10 or higher
- pip packages (see [Installation](#installation))
- A MongoDB Atlas account with the **sample_mflix** dataset loaded

Check your Python version:

```bash
python3 --version
```

---

## Installation

**1. Install dependencies:**

```bash
pip install pymongo python-dotenv protobuf dnspython grpcio grpcio-tools
```

**2. Generate the protobuf Python and gRPC files:**

```bash
python -m grpc_tools.protoc -I. --python_out=. --grpc_python_out=. movie.proto
```

This will generate `movie_pb2.py` and `movie_pb2_grpc.py` in the same directory.

---

## Environment Variables

The server reads the database password from a `.env` file.  
You must create this file before running the server.

**Create a `.env` file in the same folder as `server.py`:**

```
DB_PASSWORD=mongodb_password_here
```

The connection string uses the password to connect to MongoDB Atlas:

```
mongodb+srv://sd-red-mongodb:<password>@cluster0.nyepcaz.mongodb.net/
```

The password is URL-encoded automatically to handle special characters.

---

## How to Run

```bash
python3 server.py
```

Expected output:

```
starting gRPC server initialization...
connecting to MongoDB...
connected with MongoDB! total movies: 21349

gRPC server is listening on [::]:5000
```

The server listens on **port 5000** and accepts connections from any interface (`[::]`).  
Each client connection is handled asynchronously by the gRPC ThreadPoolExecutor.

## Supported Operations

The server exposes a `MovieService` with the following RPC methods:

| RPC Method    | Description                                                         |
| ------------- | ------------------------------------------------------------------- |
| `CreateMovie` | Inserts a new movie. Requires `title` and `year`.                   |
| `GetMovie`    | Fetches a single movie by MongoDB ID.                               |
| `UpdateMovie` | Updates a movie by ID. Requires `title` and `year`.                 |
| `DeleteMovie` | Deletes a movie by ID.                                              |
| `ListByActor` | Returns up to 20 movies matching the actor name (case-insensitive). |
| `ListByGenre` | Returns up to 20 movies matching the genre (case-insensitive).      |

Errors are returned to the client in the respective response objects with `success = false` and a descriptive `error` message.
